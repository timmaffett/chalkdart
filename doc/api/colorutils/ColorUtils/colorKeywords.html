colorKeywords property Null safety

Map<String, int> colorKeywords
final

A map of X11/CSS color keywords and their 6-digit hexadecimal RGB values x11 colors appear again here: https://www.w3.org/wiki/CSS/Properties/color/keywords complete history of the X11 color names : https://en.wikipedia.org/wiki/X11_color_names and CSS color names (derived from X11 color names): https://www.w3.org/wiki/CSS/Properties/color/keywords (these are the same as the css color list (except for X11 has 'rebeccapurple' and css colors used 'lightGoldenrodYellow' and X11 uses 'lightGoldenrod'))

  • so we include both
